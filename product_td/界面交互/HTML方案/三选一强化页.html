实现说明

1. 为什么必须独立成层

  • 升级是肉鸽的核心快感，不是战场旁边的一个小抽屉。
  • 它必须足够强打断，玩家才能感知“build 正在成型”。

2. 卡片必须表达什么

  • 类型：神通升阶 / 被动道法 / 流派联动 / 终局强化
  • 标题
  • 一句效果描述
  • 如果和当前主流派强关联，要能看出来

3. 交互逻辑

升级触发 -> 暂停战斗 -> 打开本页 点击任一强化卡 -> 立即生效 关闭本页 -> 回到原波次战斗状态 本页禁止存在“关闭但不选”的路径

4. 图片占位说明

  • 如果后续强化卡加入图标，它只承担识别功能，不应挤压文字说明。
  • AI 看到图片位时，应理解为“卡内媒体槽”，不是整卡点击区之外的额外组件。
