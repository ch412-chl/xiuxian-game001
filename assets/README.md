# 资源目录说明

本目录用于存放微信小游戏项目的正式美术资源与映射文件。

## 当前目录结构
- `妖物/完整立绘/`
- `妖物/缩略图/`
- `场景/战斗背景/`
- `界面/图标/`
- `界面/边框/`
- `界面/血条/`
- `资源映射/`

## 使用要求
- 所有自动生成或手工导入的资源都必须放入上述目录之一
- 命名需符合 `product/自动化体系/美术资源自动化规范.md`
- 代码不直接写死零散图片路径，统一通过 `资源映射` 读取
- 资源状态应与 `product/自动化体系/资源待办.md` 和各映射 JSON 保持一致
- 妖物立绘接入顺序见 `product/自动化体系/美术接入流程.md`
- 战斗背景也必须先进入 `战斗背景映射.json`，再允许代码引用
- 界面装饰、按钮边框、血条等资源也必须先进入 `界面资源映射.json`
