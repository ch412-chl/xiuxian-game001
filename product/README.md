# Product 目录索引

## 当前主入口
- `prd/prd_current.md`：当前正式 PRD
- `prd/prd_log.md`：PRD 修订日志

## 目录说明
- `prd/`：当前正式产品文档与修订记录
- `ui/`：UI/UE 规格、线框稿、开发测试检查表
- `balance/`：数值维护表、平衡分析、正式建议数值方案
- `enemies/`：敌人配置表、关卡最低通关门槛说明
- `analysis/`：玩家成长与敌我匹配分析
- `archive/`：历史版本 PRD 归档

## 推荐查看顺序
1. `prd/prd_current.md`
2. `ui/ui_ue_spec.md`
3. `balance/balance_formulas.xml`
4. `enemies/enemy_roster.csv`
5. `balance/balance_formal_proposal.md`
