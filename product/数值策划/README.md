# 数值策划目录说明

本目录只存放当前版本正式生效的数值真源。

当前职责：
- `关卡修为经验与境界升级表.csv`：关卡经验、升级经验与小境界映射真源
- `敌人配置总表.csv`：敌人运行时数值、经验奖励与敌人门槛真源
- `神通配置总表.csv`：神通名称、伤害、CD、回复效果与解锁真源
- `成就配置总表.csv`：成就目标、奖励与排序真源
- `妖物图鉴总表.csv`：图鉴档案、解锁与图鉴展示真源
- `广告配置总表.csv`：广告触发、频次限制与广告位占位真源
- `道具配置总表.csv`：丹药、符箓、装备、材料与突破道具真源
- `装备掉落配置总表.csv`：当前轻实现装备的掉落关卡与权重真源
- `破境与渡劫配置总表.csv`：突破丹消耗、成功率、冷却与渡劫表现真源
- `历练事件配置总表.csv`：探索中途事件、资源事件与陷阱事件真源
- `关卡楼层脚本总表.csv`：42 个关卡的 10 层脚本结构真源

读取建议：
1. 先读 `关卡修为经验与境界升级表.csv`
2. 再读 `敌人配置总表.csv`
3. 若实现神通 / 成就 / 图鉴 / 广告 / 道具 / 装备掉落 / 事件，则继续读取对应主表
4. 若要理解这些表如何接入代码，转到 `../实现规范/敌人与关卡脚本真源接入规范.md` 与 `../实现规范/广告系统实现级方案.md`

边界说明：
- 本目录文件是当前运行时数值真源
- 已归档的旧成长、旧难度、旧挂机方案不在此目录维护
- 当前版本若规则已进入本目录主表，代码与实现规范必须以主表为准
