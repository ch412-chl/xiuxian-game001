# 修仙版地下城堡 · 产品需求说明（当前版）

版本：v0.6.2
日期：2026-03-17
最近更新：2026-03-17 08:34:39

## 1. 产品目标
做一款“地下城堡1”核心循环的修仙题材微信小游戏。核心体验：洞府挂机产出 → 资源积累 → 角色成长 → 地下城探索 → 战利品回收 → 推进更深层。玩家通过灵仆、炼丹、装备与神通逐步变强，在地下城逐层推进并挑战首领获取关键奖励。

## 2. 平台与形态
- 平台：微信小游戏
- 视角：2D 文字+简易数值动画
- 交互：触摸点击
- 保存：本地存档（wx.setStorageSync）

## 3. 核心循环
1) 洞府挂机产出灵石与材料  
2) 洞府升级、灵仆分配、炼丹、装备与神通提升  
3) 进入地下城章节与副本，逐层推进  
4) 遇敌战斗/事件，积累战利品（暂存）  
5) 首领战胜利后结算战利品与修为  
6) 推进至下一层/下一副本

## 4. 主要系统

### 4.1 角色系统
基础属性：
- 气血（HP）
- 攻击（atk）
- 防御（def）
- 神识（spd）：影响命中率、闪避率
- 煞气：影响暴击率

煞气规则：
- 每击杀100只怪获得1点煞气
- 1点煞气 = 1%暴击率，100点煞气 = 100%暴击率
- 玩家死亡一次损失2点煞气
- 暴击率溢出100%后，每溢出1%增加1%暴击伤害（隐藏属性）

派生属性：
- totalAtk / totalDef / totalSpd（含境界加成+装备加成）

境界与修为：
- 境界列表：炼气、筑基、结丹、元婴、化神、炼虚、合体、大乘、渡劫、真仙、金仙、太乙、大罗、道祖
- 每个境界分初期、中期、后期
- xp 与 xpNeed
- 破境：修为100%时可触发突破

突破规则：
- 突破需要“突破丹”
- 突破丹材料：对应境界关卡 Boss 掉落
- 每个副本 10 关：第5关小Boss，第10关大Boss
- 首次击败 Boss 必定掉落突破材料
- 突破丹丹方为关卡内随机掉落
- 首次通关“后期副本”的小Boss必掉对应突破丹丹方
- 合成成功率100%
- 突破成功率：基础10% + 使用突破丹额外50% = 60%
- 每失败一次，基础成功率+5%，上限100%
- 突破失败惩罚：进入冷却（1小时），可观看完整广告跳过冷却

### 4.2 洞府系统
洞府等级：
- 满级10级
- 升级提升修为产出（xpPerHour）
- 提升灵仆上限：每级+10，10级共100灵仆

功能：
- 洞府升级（消耗灵石）
- 修为产出自动累积，无需领取
- 灵仆分配（矿/药园）
- 炼丹房（丹方来源见炼丹系统）
- 灵矿：安排灵仆产出灵石
- 药园：安排灵仆种植灵药

药园种子：
- 种子在关卡中随机掉落
- 种子获得后永久生效（解锁对应材料产出）

### 4.3 灵仆系统
灵仆用于洞府产出分配：
- 灵矿：产出灵石
- 药园：产出灵药
- 其他分配（待定）

灵仆仅影响洞府产出，不影响地下城战斗。

### 4.4 炼丹系统
丹方配置（RECIPES）：
- 聚灵丹（增加修为）
- 回灵丹（战斗回复气血）
- 突破丹（各境界对应丹药）

流程：
- 选择丹方 → 检查材料 → 扣材料 → 计时炼制
- 每颗丹药计时1秒
- 取消火候机制（删除文火/武火）
- 完成后自动收丹进入背包

### 4.5 背包系统
背包物品：
- 丹药（可直接使用或战斗使用）
- 材料（用于炼丹）
- 装备（法宝/法衣/符箓）
- 传送符（地下城撤离）
- 神通碎片（用于神通体系）

操作：
- 点击丹药：使用（非战斗丹药直接生效）
- 突破丹仅在修为100%且点击突破时触发使用
- 点击装备：穿戴/卸下
- 材料不可直接使用

### 4.6 装备系统
装备槽位：
- weapon（法宝）
- robe（法衣）
- ring（符箓）

装备属性加成：
- atk/def/spd
装备主要通过地下城首领奖励掉落。

### 4.8 地下城探索系统
结构设计：
- 每个境界为一个章节
- 每个章节包含3个副本（初/中/后期）
- 每个副本 10 关：5 战斗 + 5 随机事件
- 第5关小Boss，第10关大Boss
- 副本名称按主题细化（境界越高名称越高级）

关卡流程：
- 进入副本 → 逐层推进
- 事件类型：战斗、陷阱、宝箱、线索、材料
- 战利品暂存，不立即结算
- 首领战为必战

遇敌前交互：
- 展示敌人信息（气血/攻击/防御/神识/煞气、技能效果）
- 玩家可选择战斗或退出

退出逻辑：
- 不可随时撤退
- 传送符掉落概率：默认10%，每前进一层+10%（上限100%）
- 退出优先检查传送符：若有可使用带走战利品
- 若无传送符：可选择放弃战利品退出，或观看广告撤离（替代传送符）
- 选择不退出则停留在战斗前界面

死亡规则：
- 战斗失败进入死亡状态
- 可看一次广告复活（每次进入副本仅一次）
- 认命离开：本次战利品全部损失

结算规则：
- 成功撤离或击败大首领：结算战利品与修为
- 失败或认命：战利品全部损失

### 4.9 战斗表现
形式：文字战报 + 简单动画（数值跳动/轻微抖动）
战斗步骤：
- 默认玩家先手释放神通
- 命中/闪避/暴击
- 神通按已装配顺序循环释放
- 战斗日志节奏：每条战报间隔约 0.5 秒，战斗结束后停留约 2 秒
- 战斗保持单目标；如设计“群体”神通，按单体高倍率/额外伤害处理

生命恢复规则：
- 战斗结束后不回复气血
- 退出副本后气血恢复100%（包括撤离与失败）

## 7. UI 面板
底部导航：
- 角色（Role）
- 历练（Explore）
- 洞府（Cave）
- 纳戒（Bag）
- 成就（Achieve）

各面板最小可用功能：
- 角色：展示属性与装备，破境按钮
- 历练：关卡列表（分页/按境界）、探索界面、战报与战利品
- 洞府：灵仆分配、炼丹、药园、灵矿、洞府升级
- 纳戒：物品列表，丹药/装备可点击使用
- 成就：击杀/通关/境界/资源里程碑展示

神通入口：
- 神通系统整合至历练页下方，支持显示/装配/查看详情/升级
- 神通标题旁提供“！”提示，点击弹框说明“按装配顺序循环释放”
- 详情不展示冷却
- 神通详情以弹框展示；外层仅显示等级与碎片
- 所有弹框支持点击空白处关闭
- 神通装配：关卡下固定6格装配槽；下方以格子排列展示所有神通
- 点击神通列表仅查看详情，若装配槽有空位则显示“装配”
- 点击装配槽内神通弹出详情并可“移除”
- 装配槽内：名称下方仅显示等级
- 装配槽外：名称下方显示等级，等级下方显示碎片数量
- 已装配神通在列表中置灰不可点击
- 神通倍率以百分比展示

属性交互说明：
- 角色页所有属性文字带色块，可点击查看说明
- 点击煞气可查看暴击/暴击伤害溢出规则
- 属性弹框不展示说明文案（仅用于交互反馈）

探索界面规范：
- 撤离提示固定显示在底部按钮上方，避免与数值信息重叠
- 敌我信息双列对比，战斗日志居中弹窗展示
- 低明度背景 + 高对比字体，关键文本加粗；必要区域使用半透明底色块

历练列表规范：
- 三行卡片式列表，整行可点
- 左侧副本名，右侧状态
- 上一境/下一境为按钮样式
- 炼气无上一境，道祖无下一境（显示“天机未至”或空态）

## 8. 成就与称号
- 成就系统参照地下城堡1：击杀数、通关数、境界、灵石量等里程碑
- 章节终极Boss首杀授予称号并展示在角色面板

## 9. 广告规则
- 广告复活：每次进入副本仅允许一次
- 广告撤离：退出时无传送符可用（替代传送符结算）
- 广告跳过突破冷却：允许多次
- 广告位 id 需上线前替换

## 10. 失败与保护
- 防止导航点击越界导致崩溃
- 存档字段兼容旧版本

## 11. 未来扩展
- 装备稀有度/词条系统
- 地下城随机事件扩充
- 洞府灵田/灵矿独立产线深化
- 新手引导三连（首次进入/战斗/炼丹）
