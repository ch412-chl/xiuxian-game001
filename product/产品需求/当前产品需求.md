## 3. 产品目标
### 3.1 产品定位
一款以“地下城堡 1”核心循环为参考、以修仙世界观为题材的微信小游戏。

### 3.2 当前版本目标
当前版本目标不是做全量修仙系统，而是先交付一条可运行、可体验、可上线的核心链路：
- 人物页查看成长、目标与突破条件
- 历练页选择关卡与整备神通
- 探索页推进、遇敌、即时战斗、结算
- 洞府页承接成就、图鉴与阶段目标
- 纳戒 / 储物袋形成单次历练与永久库存闭环
- 境界成长、突破、广告补救、成就奖励形成最小可运营循环
- 战斗新方向进入轻肉鸽重构阶段：目标为修仙自动施法、敌潮波次、局内强化的单局生存战斗

### 3.3 当前版本不做
以下内容不纳入当前版本正式实现：
- 装备强化
- 装备词条 / 品质体系
- 洞府挂机与生产链
- 灵矿 / 灵仆 / 药园 / 丹房
- 神通碎片养成
- 十重成长
- 煞气系统
- 公会、排行、PVP、多人协作
- 新美术资源批量接入
- 完整自由移动的重型生存肉鸽第一版

## 4. 全局设计原则
### 4.1 闭环原则
每个系统都必须回答三个问题：
- 产出什么
- 消耗到哪里
- 反馈给谁

若系统只有展示、没有后续作用，则视为设计不完整。

### 4.2 单屏原则
移动端单屏优先展示当前决策所需内容，不允许堆叠重复说明、零库存信息和低优先级数值。

### 4.3 低认知成本原则
- 外层只展示当前决策必需信息
- 复杂数值留在底层结算，不强行前台展示
- 弹框只承载主界面未展示的信息

### 4.4 主链优先原则
当前版本一切优化优先服务主链：
- 人物
- 历练
- 探索
- 战斗
- 结算
- 洞府

支撑页不能反过来挤占主链决策空间。

### 4.5 真源唯一原则
当文档、代码、表格发生冲突时，不允许“按感觉折中”。必须回到正式真源判断。

## 5. 平台与技术边界
- 平台：微信小游戏
- 视角：2D 文本 + 简易 UI + 基础动画

## 7. 核心循环
### 7.1 总循环
1. 人物页查看当前境界、经验与突破条件
2. 历练页选择当前主推关卡并进入整备
3. 战前整备页确认 3 神通顺序
4. 探索推进，击败敌人即时获得修为经验与临时战利品
5. 成功撤离或通关后，将本次战利品写入纳戒
6. 返回人物页 / 洞府页查看成长、成就、图鉴与阶段目标
7. 满足条件后破境，进入下一境界主线

### 7.2 资源流
#### 历练
- 消耗：时间、续航、回灵丹、传送符
- 产出：修为经验、灵石、材料、突破丹、装备、神通使用次数

#### 储物袋
- 存放：本次历练中获得的临时战利品，以及本次可使用的回灵丹 / 传送符
- 结算：成功撤离 / 通关后写入纳戒；失败或认命离开则丢失战利品

#### 纳戒
- 存放：永久库存
- 来源：历练成功结算、成就奖励
- 去向：人物使用、突破消耗、装备穿戴

#### 洞府
- 职责：功能中枢
- 包含：成就、图鉴、阶段目标
- 不提供：挂机修为、挂机灵石、炼丹、材料生产

## 8. 全局系统规则
### 8.1 角色外显规则
外层只展示：
- 气血
- 境界
- 修为经验进度
- 是否可突破

以下属性只保留底层结算，不直接外显：
- 攻击
- 防御
- 神识
- 速度

### 8.2 境界与修为规则
- 大境界共 14 个：炼气、筑基、结丹、元婴、化神、炼虚、合体、大乘、渡劫、真仙、金仙、太乙、大罗、道祖
- 每个大境界只有 `初期 / 中期 / 后期`
- 修为经验只通过关卡内击败敌人即时获得
- 初期 / 中期满经验时自动进入下一小境界
- 后期满经验后进入可突破状态
- 突破成功后进入下一大境界初期，并清空当前经验

### 8.3 神通规则
- 每个大境界仅 1 个神通
- 最多装配 3 个神通
- 战斗中按 CD 自动释放
- 同等就绪条件下按装配顺序优先
- 神通只展示伤害与冷却，特殊效果仅保留“回复气血”

### 8.4 库存规则
- 纳戒是永久库存
- 储物袋是单次历练库存
- 0 数量项不展示
- 成功撤离 / 通关才允许将本次战利品写回纳戒
- 失败 / 认命离开会丢失本次战利品

### 8.5 成就与图鉴规则
- 成就与图鉴都从洞府进入
- 成就奖励领取后直接进入纳戒
- 妖物图鉴在首次遭遇时解锁，不要求首次击杀

### 8.6 广告规则
当前版本广告仅允许用于：
- 跳过突破失败冷却
- 战斗失败后的复活兜底
- 探索中的撤离兜底

广告不能改变主线数值真源，只能作为有限补救机制。

## 12. 存档与写回总规则
### 12.1 核心存档字段
核心存档至少包含：
- 人物成长（境界、当前经验、突破失败次数、突破冷却）
- 永久库存 `inventory`
- 单次历练库存 `runBag`
- 神通装配与成长
- 已解锁成就
- 已解锁图鉴

### 12.2 写回总原则
- 敌人死亡时：即时写入修为经验
- 探索中掉落时：先写入 `runBag`
- 成功撤离 / 通关：将 `runBag` 中的战利品写回 `inventory`
- 失败 / 认命离开：清空本次战利品写回资格
- 成就奖励领取：直接写入 `inventory`
- 突破丹消耗：只从 `inventory` 扣除
