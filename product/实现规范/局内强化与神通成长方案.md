# 局内强化与神通成长方案

## 1. 文档定位
本文档定义轻肉鸽战斗中的局内成长，不涉及局外永久成长。

## 2. 目标
局内成长必须回答：
- 为什么同一关卡打到后半段会越来越爽
- 为什么 3 个基础神通能形成不同流派
- 为什么不同局体验会变化

## 3. 成长分层
### 3.1 局外成长
- 境界
- 角色经验
- 永久神通解锁
- 永久道具积累

### 3.2 局内成长
- 神通升阶
- 新临时神通解锁
- 被动道法增益

局内成长不能写回永久数值。

## 4. 强化触发时机
推荐当前版本：
- 第一精英波结束后触发一次
- 第二精英波结束后触发一次
- 若关卡时长较长，可在最终首领前增加第三次

## 5. 三选一规则
每次强化只展示 3 个选项。

候选池必须遵守：
1. 优先出现与当前已装配神通相关的强化
2. 不允许 3 个完全无用选项同时出现
3. 不允许出现当前条件无法生效的词条
4. 同一局中应尽量避免连续 3 次都只出同一流派

## 6. 强化类型
### 6.1 神通升阶
例：
- 御剑术：飞剑数 +1
- 火弹术：爆炸半径提升
- 冰锥术：额外减速
- 天雷引：连锁数增加
- 清心诀：回复量提升

### 6.2 新临时神通
例：
- 解锁护体剑罡
- 解锁雷火余波
- 解锁寒霜环

### 6.3 被动道法
例：
- 冷却缩短
- 施法频率提升
- 范围扩大
- 穿透增加
- 吸收灵气范围提升
- 护体真气上限提升

## 7. 神通成长规则
### 7.1 基础神通数量
- 局外最多装配 3 个基础神通
- 局内临时获得的神通不计入局外装配上限

### 7.2 局内神通上限
当前版本建议：
- 同时生效的主动攻击神通不超过 5 个
- 防止画面、性能、数值同时失控

### 7.3 词条层级
每个强化词条必须标明：
- 类型：主动 / 被动 / 辅助
- 作用对象：指定神通 / 全局
- 叠加上限
- 是否允许重复

## 8. 流派目标
第一阶段建议支持的核心流派：
1. 御剑穿透流
2. 火法爆裂流
3. 冰法控场流
4. 雷法点杀流
5. 护体续航流

## 9. 真源数据要求
必须新增并维护：
- `轻肉鸽局内强化配置总表.csv`
- `轻肉鸽神通战斗配置总表.csv`

字段至少包括：
- id
- 类型
- 名称
- 作用对象
- 描述文案
- 解锁前置
- 稀有度
- 最大叠加层数
- 数值字段
- 是否允许重复出现

## 10. 依赖关系
- 依赖战斗总方案确定单局时长与强化次数
- 依赖神通配置总表提供基础神通真源
- 依赖 UI/UE 确定三选一弹层样式与展示文案

## 11. 微信运行态验收
- 强化节点出现时，战斗暂停并稳定弹出三选一
- 选择后能立即感知到战斗变化
- 同一局内不会刷出完全无用的三个选项
- 不同流派能在 2~3 次强化后产生明显手感差异
