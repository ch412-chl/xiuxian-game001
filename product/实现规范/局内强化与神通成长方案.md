# 局内强化与神通成长方案

## 1. 文档定位
本文档定义修仙轻肉鸽战斗中的**局内成长系统**。

当前正式口径：
- 每局局内等级上限 `20`
- 每次升级固定触发 `1` 次三选一强化
- 每局最多 `20` 次强化选择

本文档只定义局内成长，不定义局外永久成长。

## 2. 第一性原则
如果一局只有 2 次强化，就不是肉鸽。
真正的肉鸽成长必须满足：
1. 高频升级
2. 高频选择
3. 局内 build 持续成型
4. 后半局战斗体验明显不同于前半局

## 3. 成长分层
### 3.1 局外成长
- 境界
- 永久神通解锁
- 永久道具积累
- 长线角色经验

### 3.2 局内成长
- 局内等级提升
- 局内三选一强化
- 局内临时神通解锁
- 局内被动道法叠加

局内成长一律不写回永久成长。

## 4. 局内等级正式规则
### 4.1 等级上限
- 每局上限 `20` 级

### 4.2 升级触发
- 击杀敌人获得局内经验
- 到达升级阈值立刻暂停战斗并进入 `upgrade`
- 不允许把升级拖到精英节点统一发放

### 4.3 升级体验要求
- 前 5 级节奏偏快，用来建立初始 build
- 6~15 级为主成型期
- 16~20 级为高压补强期

## 5. 三选一正式规则
### 5.1 每次升级必给 3 选 1
- 不允许“本级无强化”
- 不允许“升级但不弹选择”
- 不允许用稀释数值代替强化选择

### 5.2 候选池类型
每次升级只能从 3 类里抽：
1. **神通升阶**：强化当前已有神通
2. **新神通解锁**：获得新的局内临时法术
3. **被动道法**：全局增益词条

### 5.3 候选生成约束
- 每次必须给 3 个可生效选项
- 优先给当前 build 相关强化
- 同时保留少量转流派机会
- 不允许连续多次全是无关流派

### 5.4 保底补齐规则
若当前层级下无法直接提供 3 个合法候选，必须按以下顺序补齐：
1. 当前流派强化保底
2. 通用被动保底
3. 低阶通用重复保底

## 6. 正式强化节奏
### 6.1 20 次强化的阶段划分
- `Lv1 ~ Lv5`：起势期
- `Lv6 ~ Lv10`：主构筑期
- `Lv11 ~ Lv15`：流派成型期
- `Lv16 ~ Lv20`：终局爆发期

### 6.2 每阶段目标
- 起势期：解决“能活下去 + 有第一手感”
- 主构筑期：形成 1 个核心主技能
- 成型期：形成技能联动与范围压制
- 终局期：针对第 20 波首领与高压怪潮补强

## 7. 正式技能门类
### 7.1 御剑系
- 御剑术
- 剑气斩
- 万剑归宗

特点：追踪、穿透、高命中、主清怪

### 7.2 火法系
- 火弹术
- 焚炎爆
- 赤阳火雨

特点：爆裂、持续灼烧、范围清群

### 7.3 冰法系
- 冰锥术
- 寒霜环
- 玄冰禁域

特点：减速、冻结、控场

### 7.4 雷法系
- 天雷引
- 雷链
- 九霄劫雷

特点：连锁、高爆发、点杀精英和首领

### 7.5 护体 / 心法系
- 护体剑罡
- 清心诀
- 回元诀

特点：生存、回复、资源续航

## 8. build 设计目标
第一版必须至少支持 5 种清晰流派：
1. 御剑穿透流
2. 火法爆裂流
3. 冰法控场流
4. 雷法点杀流
5. 护体续航流

要求：
- 每种流派在 20 次强化里都能持续成长
- 不允许 5 次后就选空
- 不允许所有流派最后都长得一样

## 10. 与波次的依赖
- 强化节奏不再绑定“第一精英后 / 第二精英后”
- 第 5 / 10 / 15 波的精英只是 build 检验点，不是强化发放点
- 理想状态是一局 20 波稳定拿满 20 次强化

## 11. 微信运行态验收
必须验证：
1. 一局内能稳定触发高频升级，而不是只升几次
2. 每次升级都弹三选一
3. 中局开始能明显感受到 build 已经成型
4. 第 15 波后 build 的伤害方式、范围和生存方式显著不同于开局
5. 20 次强化不会在中途出现大面积无效候选

## 12. 禁止事项
- 禁止继续使用“每局固定 2 次三选一”的旧口径
- 禁止把强化发放绑死在精英节点
- 禁止让局内强化污染局外 3 神通整备
- 禁止把 build 设计成单纯数值加法，没有战斗人格差异
