# 局内强化与神通成长方案

## 1. 文档定位
本文档定义修仙轻肉鸽战斗中的**局内成长系统**。

当前正式口径：
- 每局局内等级上限 `20`
- 每次升级固定触发 `1` 次三选一强化
- 每局最多 `20` 次强化选择
- 三选一卡牌只从已解锁技能的合法强化项中抽取

## 2. 第一性原则
如果一局只有 2 次强化，就不是肉鸽。
真正的肉鸽成长必须满足：
1. 高频升级
2. 高频选择
3. 局内 build 持续成型
4. 后半局战斗体验明显不同于前半局

## 3. 成长分层
### 3.1 局外成长
- 境界
- 永久技能解锁
- 永久道具积累
- 长线角色经验

### 3.2 局内成长
- 局内等级提升
- 局内三选一强化
- 已解锁技能的强化分支推进

局内成长一律不写回永久成长。

## 4. 局内等级正式规则
### 4.1 等级上限
- 每局上限 `20` 级

### 4.2 升级触发
- 击杀敌人获得局内经验
- 到达升级阈值立刻暂停战斗并进入 `upgrade`
- 不允许把升级拖到固定波次统一发放

### 4.3 升级体验要求
- 前 5 级节奏偏快，用来建立初始 build
- 6~15 级为主成型期
- 16~20 级为高压补强期

## 5. 三选一正式规则
### 5.1 每次升级必给 3 选 1
- 不允许“本级无强化”
- 不允许“升级但不弹选择”
- 不允许用稀释数值代替强化选择

### 5.2 候选池类型
每次升级的 3 张卡，只允许来自：
1. **已解锁技能强化**：强化当前已解锁技能
2. **已解锁技能分支**：同一技能的不同成长分支

### 5.3 候选生成约束
- 每次必须给 3 个可生效选项
- 只能从已解锁技能集合的合法强化项中随机抽取
- 不允许抽到未解锁技能
- 不允许因为候选不足而引入局外装配逻辑

### 5.4 保底补齐规则
若当前层级下无法直接提供 3 个合法候选，必须按以下顺序补齐：
1. 当前流派强化保底
2. 已解锁技能其他合法分支保底
3. 已解锁技能低阶重复保底

## 6. 正式强化节奏
### 6.1 20 次强化的阶段划分
- `Lv1 ~ Lv5`：起势期
- `Lv6 ~ Lv10`：主构筑期
- `Lv11 ~ Lv15`：流派成型期
- `Lv16 ~ Lv20`：终局爆发期

### 6.2 每阶段目标
- 起势期：解决“能活下去 + 有第一手感”
- 主构筑期：形成 1 个核心主技能
- 成型期：形成技能联动与范围压制
- 终局期：针对第 20 波首领与高压怪潮补强

## 7. 正式技能门类
1. 御剑系
2. 火法系
3. 冰法系
4. 雷法系
5. 护体 / 心法系

## 8. build 设计目标
第一版必须至少支持 5 种清晰流派：
1. 御剑穿透流
2. 火法爆裂流
3. 冰法控场流
4. 雷法点杀流
5. 护体续航流

要求：
- 每种流派在 20 次强化里都能持续成长
- 不允许 5 次后就选空
- 不允许所有流派最后都长得一样

## 9. 真源要求
### 9.1 必需主表
- `轻肉鸽局内等级配置总表.csv`
- `轻肉鸽神通战斗配置总表.csv`
- `轻肉鸽局内强化配置总表.csv`
- `轻肉鸽技能升级链总表.csv`

## 10. 与波次的依赖
- 强化节奏不再绑定固定精英节点
- 第 5 / 10 / 15 波的精英只是 build 检验点，不是强化发放点
- 理想状态是一局 20 波稳定拿满 20 次强化

## 11. 微信运行态验收
必须验证：
1. 一局内能稳定触发高频升级，而不是只升几次
2. 每次升级都弹三选一
3. 候选始终来自已解锁技能的合法强化项
4. 中局开始能明显感受到 build 已经成型
5. 20 次强化不会在中途出现大面积无效候选

## 12. 禁止事项
- 禁止继续使用“每局固定 2 次三选一”的旧口径
- 禁止把强化发放绑死在精英节点
- 禁止恢复局外 3 神通整备与装配逻辑
- 禁止把未解锁技能直接塞进三选一卡池
- 禁止把 build 设计成单纯数值加法，没有战斗人格差异
