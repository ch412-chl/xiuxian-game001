## 阅读顺序
1. 先读 `../产品需求/当前产品需求.md`
2. 再读 `../产品需求/模块需求导航.md`
3. 再读 `功能依赖关系总表.md`
4. 若任务属于第二轮交互与页面，优先读 `轻肉鸽交互与页面总方案.md`、`轻肉鸽页面职责与交互验收矩阵.md`、`轻肉鸽战斗边界状态机与写回规则.md`
5. 若任务属于第二轮支撑系统与运营，优先读 `轻肉鸽支撑系统映射方案.md`、`广告系统实现级方案.md`、`成就与妖物图鉴实现级方案.md`、`道具与装备轻实现级方案.md`、`破境与渡劫实现级方案.md`
