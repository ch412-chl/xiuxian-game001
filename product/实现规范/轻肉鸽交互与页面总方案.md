# 轻肉鸽交互与页面总方案

## 1. 文档定位
本文档定义修仙轻肉鸽版本的正式页面集合、页面职责、主状态承载关系、按钮显隐与返回路径。

## 2. 正式页面集合
### 2.1 主页面
1. 历练选关页
2. 技能页
3. 轻肉鸽战场页
4. 三选一强化页
5. 结算预览页

### 2.2 全屏页 / 全屏层
1. 暂停菜单层
2. 储物袋页
3. 怪物详情页
4. 撤离确认层
5. 复活确认层

## 3. 页面职责总表
| 页面 / 层 | 只负责 | 禁止承担 |
| --- | --- | --- |
| 历练选关页 | 展示当前大境界 3 关、主推信息、进入战场、进入技能页 | 展示 runBag、旧探索日志、战斗态按钮 |
| 技能页 | 展示当前已解锁技能与基础说明 | 装配、排序、结算、广告补救 |
| 轻肉鸽战场页 | 承载 `wave / boss` 实时战斗 | 展示强化卡、结算页、永久库存 |
| 三选一强化页 | 暂停战斗、提供 3 选 1 强化、确认后返回原战斗链 | 储物袋、撤离、怪物详情 |
| 结算预览页 | 展示通关 / 撤离 / 失败结果，等待进入 `settle_commit` | 直接写回纳戒、继续战斗 |
| 暂停菜单层 | 承接战斗中的次级系统入口 | 承担主战斗交互 |
| 储物袋页 | 展示本局 runBag 与战场内可用道具 | 展示永久纳戒总览 |
| 怪物详情页 | 展示当前主威胁单位信息 | 展示图鉴全量列表 |
| 撤离确认层 | 告知保留 / 丢失项，并确认是否撤离 | 直接写回 `inventory` |
| 复活确认层 | 提供广告复活或认命离开 | 广告撤离、结算写回 |

## 4. 主状态映射
| 主状态 | 所在页面 | 必须显示 | 必须隐藏 |
| --- | --- | --- | --- |
| `wave` | 轻肉鸽战场页 | 我方气血、波次、等级、经验条、暂停入口 | 强化层、结算层、储物袋常驻按钮、撤离常驻按钮 |
| `upgrade` | 三选一强化页 | 3 张强化卡、当前 build 摘要、当前波次与当前等级 | 暂停入口、储物袋、怪物详情、撤离 |
| `boss` | 轻肉鸽战场页 | 首领血条、我方状态、波次、等级、经验条、暂停入口 | 撤离、强化层、系统常驻按钮组 |
| `settle_preview` | 结算预览页 | 结果标题、经验、本局战利品或丢失说明、确认按钮 | 储物袋、怪物详情、广告撤离 |

## 5. 主链唯一口径
### 5.1 历练主链
历练选关页 -> 轻肉鸽战场页

### 5.2 技能页支链
人物页 / 历练选关页 -> 技能页 -> 返回来源页

### 5.3 20 波战斗主链
`wave(1~19) -> boss(20) -> settle_preview -> settle_commit -> leave_to_role`

### 5.4 局内升级插入规则
- 玩家在任意 `wave / boss` 中升级时，必须插入 `upgrade`
- `upgrade` 与固定波次无关
- 第 5 / 10 / 15 波只影响怪物生成，不单独切页面
- 每升 1 级固定触发 1 次 3 选 1
- 每局等级上限 20，理论最多 20 次强化选择

## 6. 页面切换规则
- 暂停菜单层：只能从 `wave / boss` 打开，打开时必须冻结战斗
- 储物袋页：只能从暂停菜单进入，打开后保持冻结战斗
- 怪物详情页：只能从暂停菜单进入
- 撤离确认层：只允许从暂停菜单进入，且仅在 `wave` 可用
- 复活确认层：只允许在 `wave / boss` 死亡后、且满足广告复活条件时打开
- 结算预览页：只允许在通关、主动撤离成功、或死亡后不复活时进入

## 7. 按钮显隐总规则
### 7.1 历练选关页
显示：关卡卡片、主推标签、进入战斗按钮、技能页入口、人物 / 洞府导航
隐藏：runBag、储物袋、怪物详情、旧探索按钮、旧自动推进按钮

### 7.2 技能页
显示：已解锁技能列表、技能详情、返回来源页
隐藏：装配按钮、顺位槽、开始战斗按钮、runBag、结算、广告按钮

### 7.3 轻肉鸽战场页
- `wave`：显示 `暂停入口`
- `boss`：显示 `暂停入口`
- 统一隐藏：常驻储物袋、常驻怪物详情、常驻撤离、结算按钮、强化卡、旧“遇敌待战”按钮组

### 7.4 三选一强化页
显示：3 张强化卡、当前 build 摘要
补充：候选只允许从已解锁技能的合法强化项中抽取；若不足 3 项，必须从已解锁技能的其他合法分支补齐到 3 项。
隐藏：暂停入口、所有次级系统入口、返回、关闭

### 7.5 怪物详情页
- 普通怪：名称、类型标签、1 条机制摘要
- 精英 / 首领：名称、类型标签、当前气血、1~2 条技能摘要、技能伤害与 CD（真源存在时）

## 8. 数据依赖
- 历练选关页：`轻肉鸽关卡映射总表.csv`
- 技能页：`神通配置总表.csv`、`轻肉鸽神通战斗配置总表.csv`、`轻肉鸽技能升级链总表.csv`
- 战场相关：`轻肉鸽敌潮波次配置总表.csv`、`轻肉鸽敌群组成总表.csv`、`轻肉鸽精英配置总表.csv`、`轻肉鸽首领机制配置总表.csv`、`轻肉鸽掉落组配置总表.csv`、`轻肉鸽局内等级配置总表.csv`

## 9. 写回依赖
- runBag 只在战场中累积
- 经验在本局结束后进入 `settle_commit` 写回人物经验
- 战利品只允许在 `settle_commit` 写入 `inventory`
- 失败认命离开时，经验保留，runBag 战利品丢失
- 广告复活成功时，runBag、局内强化与死亡前主状态保留

## 10. 当前禁止事项
1. 禁止恢复战前整备页、局外装配页或 3 槽出手顺序逻辑
2. 禁止把 runBag 展开在战场主视图下半区
3. 禁止在 `boss` 状态显示撤离常驻入口
4. 禁止把三选一强化做成侧边小弹框
5. 禁止把旧“遇敌待战”按钮组直接套到轻肉鸽战场页
6. 禁止把广告复活 / 广告撤离继续挂在失败预览页上
7. 禁止再假设玩家依赖移动走位完成战斗
