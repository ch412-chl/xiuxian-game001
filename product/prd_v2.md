# 修仙版地下城堡 · 产品需求说明

版本：v0.1
日期：2026-03-17
最近更新：2026-03-17 04:49:22

## 1. 产品目标
做一款“地下城堡1”核心循环的修仙题材微信小游戏。核心体验是：洞府挂机产出 → 资源积累 → 角色成长 → 地下城探索 → 战利品回收 → 推进更深层。玩家通过洞府灵仆、炼丹、装备与神通逐步变强，在地下城逐层推进并挑战首领获取关键奖励。

## 2. 平台与形态
- 平台：微信小游戏
- 视角：2D 文字+简易数值动画
- 交互：触摸点击
- 保存：本地存档（wx.setStorageSync）

## 3. 核心循环
1) 洞府挂机产出灵石与材料  
2) 洞府升级、炼丹、装备与神通提升  
3) 进入地下城关卡，逐层推进  
4) 遇敌战斗/事件，积累战利品（暂存）  
5) 首领战胜利后结算战利品与修为  
6) 继续下一关卡

## 4. 主要系统

### 4.1 角色系统
基础属性：
- 气血
- 攻击（atk）
- 防御（def）
- 神识（spd），影响命中率、闪避率
- 煞气， 影响暴击率（只能通过击杀怪物获得，每击杀100只怪获得1点煞气，1点煞气=1%暴击率，100点煞气=100%暴击率，如玩家死亡一次则损失2点煞气）

派生属性：
- totalAtk / totalDef / totalSpd（含境界加成+装备加成）
- power（战力展示用：totalAtk*2 + totalDef + totalSpd*0.5）

境界与修为：
- realmIdx 对应境界列表
炼气、筑基、结丹、元婴、化神、炼虚、合体、大乘、渡劫、真仙、金仙、太乙、大罗、道祖
每个境界分初期、中期、后期

- xp 与 xpNeed
- 破境：修为足够时可尝试突破
每个境界后期修为100%时，可选择突破至下个境界。
突破境界需要突破丹，突破丹材料为对应境界关卡的Boss掉落，每个关卡设计10关，第5关为小boss,第10关为大boss，首次击败关卡boss时必定掉落该突破材料，突破丹丹方为关卡内随机掉落，突破丹合成概率100%。
默认突破境界时的成功率为10%，使用突破丹提高50%概率，总共60%，每突破失败一次，基础概率提高5%，下次使用突破丹后总概率为65%，上限100%。
- 失败惩罚：突破失败仅限制下次突破CD，如第一次突破失败，需要等待1小时，方可进行下次突破，但观看完整广告可立即结束冷却时间，如多次突破失败，可看多次，不限制。
- 突破丹药加成：筑基丹、结丹丹提升突破率，突破丹命名方式以境界类推（例如筑基丹、结丹丹、元婴丹、化神丹、炼虚丹...）

### 4.2 洞府系统
洞府等级（cave.lv）影响：
- 洞府满级10级
- 修为产出，升级洞府增加修为产出效率（xpPerHour）
- 灵仆数量上限，每升级洞府增加10灵仆数量，10级洞府共100灵仆
- 挂机材料产出基准

功能：
- 洞府升级（消耗灵石）
- 洞府提供的修为无需领取，直接加到修为上
- 灵仆
- 炼丹房（抄写凡人修仙传中的一些丹方，生成一些）
- 灵矿，需要安排灵仆才能产出灵石收益
- 药园，需要安排灵仆种植照顾灵药，灵药种子在关卡中随机掉落，种子获取后即永久生效（抄写凡人修仙传中的一些丹方材料，生成一些种子）

### 4.3 灵仆系统
灵仆为洞府挂机的增益选择：
- 灵矿：产出灵石
- 药园：产出灵药
- 其他待定

灵仆只影响洞府挂机产出，不影响地下城战斗。

### 4.4 炼丹系统
丹方配置（RECIPES）：
示例：
- 聚灵丹（增加修为）
- 回灵丹（战斗回复气血）
- 筑基丹/结丹丹（突破加成概率）

流程：
- 选择丹方 → 检查材料 → 扣材料 → 每颗丹药计时1秒
- 删除火候
- 完成后自动收丹进入背包

### 4.5 背包系统
背包物品：
- 丹药（可直接使用或战斗使用）
- 材料（用于炼丹）
- 装备（武器/法衣/戒指）
- 传送符（地下城撤离）
- 神通碎片（用于神通体系）

操作：
- 点击丹药：使用（非战斗丹药直接生效，突破丹只能在修为100%时在修完进度条处点击触发使用）
- 点击装备：穿戴/卸下
- 材料不可直接使用

### 4.6 装备系统
装备槽位：
- weapon（法宝）
- robe（法衣）
- ring（符箓）

装备带属性加成：
- atk/def/spd
装备只通过地下城首领奖励掉落（当前版本如此）。

### 4.7 神通系统
神通可装配（最多6个），战斗中按顺序循环释放。
神通碎片在地下城探索关卡中掉落。
升级机制在数据层定义，提供升级。
升级需要消耗灵石+符合条件的神通碎片。
点击神通可查看神通数据（倍率+效果）

### 4.8 地下城探索系统
关卡列表：
- 按顺序解锁（通关前一关后解锁下一关）
- 境界不限制进入（规则确认）
- 每个境界分1个章节3个副本，如野狼谷外围=炼气初期、野狼谷中心=炼气中期、野狼谷深处=炼气后期，后续以此类推

每关流程：
- 进入关卡 → 逐层推进（每个副本默认5个关卡战斗+5个非战斗类随机事件，第5关和第10关为小、大首领站）
- 事件类型：战斗、陷阱、宝箱、线索、材料
- 战利品暂存，不立即结算
- 首领战为必战
- 遇敌战斗前，可查看敌人信息（气血/攻击/防御/神识/煞气，技能效果），玩家可选择战斗或退出，如选择退出判断是否有传送符，如有传送符询问玩家是否使用传送符，如选择是，则携带当前战利品退出副本，如选择不回，则停留在战斗前界面。如选择退出时没有传送符，询问玩家是否放弃当前战利品退出战斗，下方另提供观看广告选项（替代传送符回城），如选择是则损失当前关卡已获得战利品退出副本，如选择否则停留在战斗前界面。如选择战斗则开始战斗

退出规则：
- 不可随时撤退
- 本关随机掉落 1 次传送符（仅一次）
- 使用传送符可撤离结算
- 可看广告撤离结算

死亡规则：
- 战斗失败进入死亡状态
- 可看一次广告复活（每次进入关卡仅一次）
- 认命离开：本次战利品全部损失

结算规则：
- 成功撤离或击败首领：结算战利品与修为
- 失败或认命：战利品全部损失

### 4.9 战斗表现
形式：文字战报 + 简单动画（数值跳动/轻微抖动）
战斗步骤：
- 默认玩家先放神通
- 命中/闪避/暴击
- 神通按玩家已装配神通依次按顺序循环释放，直到战斗结束，玩家或敌人一方气血归0
- 战斗结束后，不回复气血，保持战斗结束时的气血数值。仅关卡最后大首领战斗胜利退出关卡后回复100%气血

## 5. 数据结构（存档字段）
需要的核心字段（示例）：
- name, realmIdx, xp, stone
- atk, def, spd
- equip: { weapon, robe, ring }
- bag: [{ id, count }]
- shengtong: [{ id, lv, frag }]
- equippedSt: [id...]
- cave: { lv, alchemyStart, alchemyRecipeId, alchemyFireLv, idle: { startTime, servantId } }
- maxStageCleared
- lastOnline

## 6. 关键数值与平衡（可调整）
- 洞府产出与升级成本见 CAVE_LEVELS
- 关卡奖励见 STAGES.rewards
- 灵仆见 SERVANTS
- 装备属性见 EQUIPS
- 丹方耗材/时间/效果见 RECIPES

## 7. UI 面板
底部导航：
- 角色（Role）
- 神通（St）
- 历练（Explore）
- 洞府（Cave）
- 锦囊修改为纳戒（Bag）

各面板最小可用功能：
- 角色：展示属性与装备，破境按钮
- 神通：显示与装配
- 历练：关卡列表、探索界面、战报与战利品
- 洞府：灵仆选择、炼丹、药园、灵矿、洞府升级
- 锦囊：物品列表，丹药/装备可点击使用

## 8. 广告规则
- 广告复活：每次进入关卡仅允许一次
- 广告撤离：任何探索中可用（退出结算）
- 广告位 id 需在上线前替换

## 9. 失败与保护
- 防止导航点击越界导致崩溃
- 存档字段兼容旧版本
