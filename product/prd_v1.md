# 修仙版地下城堡 · 产品需求说明

版本：v0.1
日期：2026-03-17
最近更新：2026-03-17 04:49:22

## 1. 产品目标
做一款“地下城堡1”核心循环的修仙题材微信小游戏。核心体验是：洞府挂机产出 → 资源积累 → 角色成长 → 地下城探索 → 战利品回收 → 推进更深层。玩家通过洞府灵仆、炼丹、装备与神通逐步变强，在地下城逐层推进并挑战首领获取关键奖励。

## 2. 平台与形态
- 平台：微信小游戏
- 视角：2D 文字+简易数值动画
- 交互：触摸点击
- 保存：本地存档（wx.setStorageSync）

## 3. 核心循环
1) 洞府挂机产出灵石与材料  
2) 洞府升级、炼丹、装备与神通提升  
3) 进入地下城关卡，逐层推进  
4) 遇敌战斗/事件，积累战利品（暂存）  
5) 首领战胜利后结算战利品与修为  
6) 继续下一关卡

## 4. 主要系统

### 4.1 角色系统
基础属性：
- 攻击（atk）
- 防御（def）
- 速度（spd）

派生属性：
- totalAtk / totalDef / totalSpd（含境界加成+装备加成）
- power（战力展示用：totalAtk*2 + totalDef + totalSpd*0.5）

境界与修为：
- realmIdx 对应境界列表
- xp 与 xpNeed
- 破境：修为足够时可尝试突破
- 失败惩罚：修为衰减至70%
- 丹药加成：筑基丹、结丹丹提升突破率

### 4.2 洞府系统
洞府等级（cave.lv）影响：
- 修为产出（xpPerHour）
- 灵石产出（stonePerHour）
- 挂机材料产出基准

功能：
- 洞府升级（消耗灵石）
- 洞府挂机收益领取
- 灵仆选择影响挂机收益

### 4.3 灵仆系统
灵仆为洞府挂机的增益选择：
- 矿师：灵石更多、材料略少
- 药童：材料更多、灵石略少、稀有材料更多
- 杂役：均衡

灵仆只影响洞府挂机产出，不影响地下城战斗。

### 4.4 炼丹系统
丹方配置（RECIPES）：
示例：
- 聚灵丹（增加修为）
- 回灵丹（战斗回复）
- 筑基丹/结丹丹（突破加成）

流程：
- 选择丹方 → 检查材料 → 扣材料 → 开始计时
- 火候：文火/武火（武火缩短时间）
- 完成后可收丹进入背包

### 4.5 背包系统
背包物品：
- 丹药（可直接使用或战斗使用）
- 材料（用于炼丹）
- 装备（武器/法衣/戒指）
- 传送符（地下城撤离）
- 神通碎片（用于神通体系）

操作：
- 点击丹药：使用（非战斗丹药直接生效）
- 点击装备：穿戴/卸下
- 材料不可直接使用

### 4.6 装备系统
装备槽位：
- weapon（兵器）
- robe（法衣）
- ring（戒指）

装备带属性加成：
- atk/def/spd
装备只通过地下城首领奖励掉落（当前版本如此）。

### 4.7 神通系统
神通可装配（最多6个），战斗中按顺序循环释放。
神通碎片在地下城探索关卡中掉落。
升级机制在数据层定义，但当前 UI 不提供升级入口（后续扩展）。

### 4.8 地下城探索系统
关卡列表：
- 按顺序解锁（通关前一关后解锁下一关）
- 境界不限制进入（规则确认）

每关流程：
- 进入关卡 → 逐层推进（层数由关卡定义或默认公式）
- 事件类型：战斗、陷阱、宝箱、线索、材料
- 战利品暂存，不立即结算
- 首领战为必战

退出规则：
- 不可随时撤退
- 本关随机掉落 1 次传送符（仅一次）
- 使用传送符可撤离结算
- 可看广告撤离结算

死亡规则：
- 战斗失败进入死亡状态
- 可看一次广告复活（每次进入关卡仅一次）
- 认命离开：本次战利品全部损失

结算规则：
- 成功撤离或击败首领：结算战利品与修为
- 失败或认命：战利品全部损失

### 4.9 战斗表现
形式：文字战报 + 简单动画（数值跳动/轻微抖动）
战斗步骤：
- 按速度先后轮流
- 命中/闪避/暴击
- 神通概率释放

## 5. 数据结构（存档字段）
需要的核心字段（示例）：
- name, realmIdx, xp, stone
- atk, def, spd
- equip: { weapon, robe, ring }
- bag: [{ id, count }]
- shengtong: [{ id, lv, frag }]
- equippedSt: [id...]
- cave: { lv, alchemyStart, alchemyRecipeId, alchemyFireLv, idle: { startTime, servantId } }
- maxStageCleared
- lastOnline

## 6. 关键数值与平衡（可调整）
- 洞府产出与升级成本见 CAVE_LEVELS
- 关卡奖励见 STAGES.rewards
- 灵仆倍率见 SERVANTS
- 装备属性见 EQUIPS
- 丹方耗材/时间/效果见 RECIPES

## 7. UI 面板
底部导航：
- 角色（Role）
- 神通（St）
- 历练（Explore）
- 洞府（Cave）
- 锦囊（Bag）

各面板最小可用功能：
- 角色：展示属性与装备，破境按钮
- 神通：显示与装配
- 历练：关卡列表、探索界面、战报与战利品
- 洞府：灵仆选择、挂机领取、炼丹、洞府升级
- 锦囊：物品列表，丹药/装备可点击使用

## 8. 广告规则
- 广告复活：每次进入关卡仅允许一次
- 广告撤离：任何探索中可用（退出结算）
- 广告位 id 需在上线前替换

## 9. 失败与保护
- 防止导航点击越界导致崩溃
- 存档字段兼容旧版本

## 10. 未来扩展
- 神通碎片升级 UI
- 装备稀有度/词条系统
- 地下城随机事件扩充
- 洞府灵田/灵矿独立产线
