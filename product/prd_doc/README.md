# 产品需求

## 目录职责
本目录只保留产品总控层文档：
- `current_prd.md`：当前版本总控 PRD
- `module_map.md`：产品模块地图与版本边界
- `changelog.md`：近期有效变更记录
- `prd_template.md`：模块级需求模板

## 阅读顺序
1. 先读 `current_prd.md`
2. 再读 `module_map.md`
3. 只有需要追最近变更时，再读 `changelog.md`

## 说明
- 本目录不放实现级方案
- 本目录不放字段级主表说明
- 更早修订历史已归档到 `../历史归档/修订日志归档/`
- 若任务涉及具体实现，请离开本目录，进入 `../实现规范/`
- 若任务涉及数值和配置，请进入 `../数值策划/`
