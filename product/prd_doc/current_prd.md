## 1. 文档定位
本文档只负责定义当前版本的产品设计：
- 当前版本要做什么
- 当前版本不做什么
- 核心玩法与核心体验是什么
- 各模块在产品层的职责和边界是什么

本文档不承载：

## 2. 产品定位
### 2.1 产品基准
- 世界观：修仙
- 战斗骨架：**对标《向僵尸开炮》**
- 当前正式方向：**修仙塔防式自动施法肉鸽**

### 2.2 当前版本目标
交付一条可运行、可体验、可上线的战斗主链：
- 人物页查看成长、主推目标与突破条件
- 历练页选择关卡并完成战前整备
- 进入单局战场后承载 20 波敌潮、20 级、20 次强化的自动施法战斗
- 洞府页承接成就、图鉴与阶段目标
- 纳戒 / 储物袋形成单次历练与永久库存闭环
- 境界成长、突破、广告补救、成就奖励形成最小可运营循环

### 2.3 当前版本不做
- 十重成长
- 煞气系统
- 洞府挂机与生产链
- 自由移动式生存肉鸽
- 新美术资源批量接入

## 3. 当前版本正式战斗口径
1. 每局固定 **20 波**
2. 每局局内等级上限 **20 级**
3. 每升 1 级固定触发 **1 次三选一强化**，理论最多 **20 次强化**
4. 第 5 / 10 / 15 波为精英波
5. 第 20 波为首领终局
6. 战斗体验必须对齐《向僵尸开炮》：敌潮持续压近、玩家固定底部自动攻击、局内高频成长、build 在一局内明显成型

## 4. 全局设计原则
1. 主链优先：一切优化优先服务人物 -> 历练 -> 战场 -> 结算 -> 洞府
2. 战场优先：战场页必须以实时战斗体验为主，不能再做成页式探索换皮
3. 低认知成本：外层只展示当前决策必须信息
4. 资源闭环：每个系统都必须回答“产出什么、消耗到哪里、反馈给谁”
5. 题材一致：战斗骨架对标《向僵尸开炮》，表现语言必须修仙化，而不是枪械换皮

## 5. 核心循环
1. 玩家在人物页查看当前境界、经验与突破条件
2. 进入历练页选择当前主推关卡
3. 在战前整备页确认局外基础神通与出手顺序
4. 进入 20 波单局战场，玩家固定在底部防线自动攻击敌潮
5. 击杀敌人获得局内经验，升级时进行三选一卡牌强化
6. 成功撤离或通关后，本次战利品写回纳戒；失败则只保留经验，不保留本次战利品
7. 返回人物页 / 洞府页查看成长、成就、图鉴与阶段目标
8. 满足条件后破境，进入下一境界主线

## 6. 核心模块职责
### 6.1 人物页
职责：展示当前成长状态，并给出下一步行动引导。
必须回答：
- 当前境界到哪里了
- 是否可突破
- 当前主推关卡是什么
- 当前主要阻塞是什么

### 6.2 历练页
职责：承接主线推进入口。
必须回答：
- 当前该打哪一关
- 当前推荐战前整备是什么
- 哪些关卡已解锁 / 未解锁

### 6.3 战前整备页
职责：完成局外基础 build 确认。
必须回答：
- 当前带哪 3 个局外基础神通
- 出手顺序是什么
- 进入战斗前是否准备完毕

### 6.4 单局战场页
职责：承载当前版本的主体验。
必须满足：
- 玩家固定在底部防线，不进行自由移动
- 神通自动攻击范围内敌人
- 敌潮持续压近，形成波次压力
- 升级三选一是局内唯一高频决策
- 非战斗功能不允许抢主视觉

### 6.5 结算页
职责：收束单局结果，并把收益反馈给长期成长。
必须回答：
- 这一局得到了什么
- 哪些收益写回长期成长
- 下一步推荐继续什么

### 6.6 洞府页
职责：作为轻中枢页承接目标、成就、图鉴。
不再承担：
- 挂机
- 生产
- 洞府升级

## 7. 全局系统规则
### 7.1 角色外显规则
外层只展示：
- 气血
- 境界
- 修为经验进度
- 是否可突破

### 7.2 境界与修为规则
- 大境界共 14 个
- 每个大境界只有 `初期 / 中期 / 后期`
- 修为经验只通过关卡内击败敌人即时获得
- 后期满经验后进入可突破状态

### 7.3 神通规则
- 每个大境界仅 1 个核心神通
- 最多装配 3 个局外基础神通
- 战斗中自动释放
- 局内通过升级和强化形成临时 build

### 7.4 库存规则
- 纳戒是永久库存
- 储物袋是单次历练库存
- 成功撤离 / 通关才允许将本次战利品写回纳戒
- 失败 / 认命离开会丢失本次战利品，但经验保留

### 7.5 成就与图鉴规则
- 成就与图鉴都从洞府进入
- 成就奖励领取后直接进入纳戒
- 妖物图鉴在首次遭遇时解锁，不要求首次击杀

### 7.6 广告规则
当前版本广告仅允许用于：
- 战斗失败后的复活兜底
- 探索中的撤离兜底
- 跳过突破失败冷却

## 8. 当前版本验收目标
### 8.1 产品层验收
1. 玩家能理解“当前该做什么”
2. 战斗体验明确对齐《向僵尸开炮》的塔防式自动攻击骨架
3. 单局内成长感明显，不会出现“打完全局只强化两次”的伪肉鸽体验
4. 战场主视觉清楚，非战斗功能不喧宾夺主
5. 长期成长、单局成长、结算反馈三者能形成闭环

### 8.2 版本阻塞项
当前版本上线前仍必须补齐：
1. 代码按正式真源接入
2. 微信开发者工具运行态回归通过
3. 广告真实 `adUnitId` 替换完成
