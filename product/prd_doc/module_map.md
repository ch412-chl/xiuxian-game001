# 模块需求导航

## 1. 文档定位
本文档是当前版本的产品模块地图，只回答三个问题：
1. 当前有哪些正式模块
2. 每个模块在产品层解决什么问题
3. 每个模块的当前版本边界是什么

若需要具体实现、状态机、写回规则、字段或主表，请离开本目录，进入：
- `../实现规范/`
- `../数值策划/`
- `../界面交互/`

## 2. 当前唯一主线
当前唯一产品主线：**修仙塔防式自动施法肉鸽，对标《向僵尸开炮》**。

旧页式主链（选层推进、日志探索、单体结算）已归档，不再作为当前版本产品目标。

## 3. 当前正式模块
| 模块 | 用户目标 | 核心体验 | 当前版本边界 |
| --- | --- | --- | --- |
| 人物页 | 看懂当前成长状态与下一步目标 | 明确当前境界、经验、突破条件、主推关卡 | 不做深背包管理，不承载图鉴与成就入口 |
| 历练页 | 明确当前该打哪一关 | 选关清晰、主推明确、进入整备顺畅 | 不承担局内升级，不承担复杂战斗操作 |
| 战前整备页 | 开局前确认基础 build | 选择 3 个局外基础神通与出手顺序 | 不在这里做局内成长，不做复杂背包操作 |
| 单局战场页 | 体验当前版本核心爽点 | 玩家固定底部自动攻击，20 波推进，20 级成长，20 次三选一强化 | 不做自由移动，不做重型手操技能释放 |
| 三选一强化 | 在局内持续做成长决策 | 高频升级、流派成型、构筑爽感 | 不做战外长期养成 |
| 结算页 | 理解一局收益和长期反馈 | 看到收益、损失、长期成长变化 | 不承担战斗主体验 |
| 洞府页 | 查看长期目标和支撑系统 | 阶段目标、成就、图鉴、进度反馈 | 不恢复挂机和生产链 |
| 成就页 | 获得长期目标反馈 | 进度、奖励、收集感 | 不承载主战斗功能 |
| 妖物图鉴页 | 理解敌人档案与收集反馈 | 遭遇解锁、档案查看、题材沉浸 | 不作为战斗中主操作 |
| 广告补救系统 | 提供有限兜底 | 失败复活、撤离兜底、跳冷却 | 不允许变成主收益来源 |
| 破境系统 | 驱动长期成长跃迁 | 达成条件后进入下一阶段 | 不做复杂赌脸玩法第一版 |

## 4. 当前版本模块关系
### 4.1 主链模块
人物页 -> 历练页 -> 战前整备页 -> 单局战场页 -> 结算页 -> 人物页 / 洞府页

### 4.2 支撑模块
- 洞府页承接：成就、图鉴、阶段目标
- 破境系统承接：人物成长后的阶段跃迁
- 广告系统承接：极少数失败兜底与冷却跳过

## 5. 当前版本最重要的产品边界
1. 战场页是唯一主体验，不再允许做成旧页式探索换皮
2. 玩家在战场内不移动，主决策是升级时的三选一卡牌强化
3. 洞府、成就、图鉴都是支撑模块，不得反客为主
4. 旧挂机、生产、煞气、十重成长均不再进入当前版本
