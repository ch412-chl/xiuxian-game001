# 模块需求导航

## 1. 文档定位
本文档只回答三个问题：
1. 当前有哪些正式模块
2. 每个模块解决什么用户问题
3. 各模块如何衔接主线

## 2. 当前唯一主线
当前唯一产品主线：**修仙塔防式自动施法肉鸽，对标《向僵尸开炮》**。

## 3. 当前正式模块
| 模块 | 用户目标 | 核心体验 |
| --- | --- | --- |
| 人物页 | 看懂当前成长状态与下一步目标 | 当前境界、经验、突破条件、主推关卡 |
| 历练页 | 明确当前该打哪一关 | 选关清晰、主推明确、直接进战场 |
| 技能页 | 看懂自己已解锁的技能 | 技能认知清晰，不做装配 |
| 单局战场页 | 体验当前版本核心爽点 | 底部固定防线、自动施法、20 波敌潮、20 次强化 |
| 三选一强化 | 在局内持续做成长决策 | 高频升级、build 成型 |
| 结算页 | 理解一局收益和长期反馈 | 看到收益、损失、长期成长变化 |
| 洞府页 | 查看长期目标和支撑系统 | 阶段目标、成就、图鉴、进度反馈 |
| 成就页 | 获得长期目标反馈 | 进度、奖励、收集感 |
| 妖物图鉴页 | 理解敌人档案与收集反馈 | 遭遇解锁、档案查看 |
| 广告补救系统 | 提供有限兜底 | 失败复活、撤离兜底、跳冷却 |
| 破境系统 | 驱动长期成长跃迁 | 达成条件后进入下一阶段 |

## 4. 模块关系
- **主链**：人物页 -> 历练页 -> 单局战场页 -> 结算页 -> 人物页 / 洞府页
- **辅助链**：人物页 / 历练页 -> 技能页 -> 返回来源页
- **支撑链**：洞府页 -> 成就页 / 妖物图鉴页

## 5. 当前产品边界
1. 战场页是唯一主体验，不再允许做成旧页式探索换皮。
2. 玩家在战场内不移动，主决策是升级时的三选一卡牌强化。
3. 技能页只负责查看已解锁技能，不承担装配和排序。
4. 洞府、成就、图鉴都是支撑模块，不得反客为主。
