# 页面交互导航

## 1. 文档定位
本文档用于回答三件事：
- 这个页面现在属于哪条主线
- 该读哪份实现规范
- 该看哪份验收文档

它不承担：
- 总体 UI 原则
- 数值真源说明
- 完整状态机全文

## 3. 页面级阅读顺序

2. 本文档
3. 对应实现规范
4. `界面交互线框.html`

### 3.2 做前端实现时
1. `../prd_doc/current_prd.md`

4. 本文档
5. 对应实现规范
6. `../实现规范/实现前自检流程.md`
7. 对应运行态验收文档

### 3.3 做微信运行态验收时
1. `界面交互规范.md`
2. 本文档
3. 对应实现规范
4. `../实现规范/微信开发者工具运行态验收总表.md`
5. 若属于轻肉鸽战斗链，再补 `../实现规范/轻肉鸽微信运行态验收补充.md`

## 4. 使用提醒
- 本文档是“页面入口地图”，不是详细交互全文。
- 若某页面涉及状态机、写回规则或数据字段，请转到对应实现规范，不要只凭本页导航猜实现。
- 若页面修改会影响交互体验，仍需先给方案，再进代码实现。
