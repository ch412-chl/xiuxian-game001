# 页面交互导航

## 1. 文档定位
本文档用于把“页面该读哪份文档、看哪张线框、依赖哪份实现规范”快速收口。

它不承担：
- 总体 UI 原则
- 数值真源说明
- 完整状态机全文

使用方式：
1. 先读 `界面交互规范.md`
2. 再在本文档中定位目标页面
3. 进入对应实现规范与线框
4. 最后对照微信运行态验收总表

## 4. 使用提醒
- 本文档是“页面入口地图”，不是详细交互全文。
- 若某页面涉及状态机、写回规则或数据字段，请转到对应实现规范，不要只凭本页导航猜实现。
- 若页面修改会影响交互体验，仍需先给方案，再进代码实现。
