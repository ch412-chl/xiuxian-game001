实现说明

1. 第一眼必须看到什么

  • 敌潮和战场中央，而不是按钮。
  • 顶部 HUD 是“辅助读数”，不是主视觉中心。
  • 底部条是状态与次级操作，不是操作面板主场。

2. UI 层级

第一层：战场核心层（敌潮 / 角色 / 术法） 第二层：状态层（波次 / 等级 / 经验 / 气血 / 首领血条） 第三层：次级操作层（储物袋 / 怪物详情 / 撤离） 第四层：中断层（升级 / 首领过场 / 复活 / 结算）

3. 关键显隐

  • `wave / elite`：显示储物袋、怪物详情、撤离。
  • `boss`：显示储物袋、怪物详情，隐藏撤离。
  • `upgrade`：隐藏所有底部按钮，只留升级弹层。

4. 图片占位如何理解

  • 后续战场背景图是“大氛围层”，不是可点区域。
  • 首领立绘 / 敌潮剪影属于战场表现层，不应当替代真实碰撞或逻辑区域。
  • 如果 HTML 里出现图片占位，AI 实现时应把它当作层级与用途说明，而不是只当静态装饰。

5. 对齐要求

要像《向僵尸开炮》的地方： - 战场是主视觉 - 升级是高频中断 - 敌潮压迫持续存在 不能照抄的地方： - 枪械语义 - 军武风 UI - 现代末世词汇
