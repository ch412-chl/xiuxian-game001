轻肉鸽页面 HTML 方案总览

本目录用于承载当前主线页面的 HTML 交互方案稿。每个页面都同时包含：结构草图、状态说明、图片占位用途、显隐规则、数据依赖与实现提醒。 目标不是“画得好看”，而是让实现线程直接看懂页面应该如何落地。

主链入口

轻肉鸽历练选关页

用于定义主推关卡、进入整备、境界阶段提示与主导航关系。

  • 只做选关，不做神通详细管理
  • 主推关卡必须一眼可见
  • 禁止恢复旧日志探索结构

用于定义 3 神通顺位、待解锁槽位、确认进入战场的行为与状态说明。

  • 只负责整备，不负责选关
  • 槽位主导，不允许神通列表抢主视觉
  • 未解锁槽位使用固定文案占位

  • 战场是绝对主视觉
  • 升级、首领、死亡采用中断层
  • 底部按钮只可用，不可抢眼
